Model Parameter Design

• We restructured the neural network using highly efficient, hardware-compatible transformer elements.
• This design maximizes representational power while adhering to the 2,000,000 parameter limit.

Weight Parameter Tying
We linked lm_head and token embedding weights to reclaim approximately 370,000 parameters, which we allocated to deeper layers and wider attention dimensions.
Rotary Relative Ordering (RoPE)
We integrated RoPE calculations directly into the SelfAttention queries and keys, which replaced fixed lookup tables and improved sequence length generalization.
RMSNorm and SwiGLU Activation
We substituted RMSNorm for LayerNorm to avoid calculating mean values, and we swapped GELU for SwiGLU gates in the feedforward networks.
Layer and Channel Configuration
We set the network shape to 4 layers, 6 attention heads, and a 180-unit embedding width, which established exactly 1,928,340 parameters.
